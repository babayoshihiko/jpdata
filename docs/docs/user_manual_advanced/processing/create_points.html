7.3. 人口の数だけ点を作成

国勢調査には人口 (JINKO) という属性があります。

この属性を使って、人口の数だけ点を作成するアルゴリズムを実行します。

まず、プラグイン jpdata を使って、国勢調査のデータをダウンロードしましょう。

次に、プロセシングで points などと検索し、「ポリゴン内にランダム点群 (inside)」を探します。

この画面で、少しわかりづらいですが、「点の数もしくは密度」の右側にあるアイコンをクリックします。

ここで、JINKO を選択します。

実行を行うことで、点が作成されます。

作成する点の多さに応じて時間がかかります。
