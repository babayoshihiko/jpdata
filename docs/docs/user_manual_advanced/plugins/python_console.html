8.4.1. コードエディタ

Show Editor ボタンを押してコードエディタを表示します。

フォルダ内の zip ファイルをすべて展開するコードです。

import os, zipfile
zip_dir = "C:/Users/baba/Desktop/PackDLMap"
dest_dir = " C:/Users/baba/Desktop/PackDLMap "
for file in os.listdir(zip_dir):
  fullpath = os.path.join(zip_dir, file)
  if zipfile.is_zipfile(fullpath):
    with zipfile.ZipFile(fullpath) as item:
      item.extractall(dest_dir)

すべての RdEdg を別フォルダに移動（RdEdgフォルダをあらかじめ作っておく）

import glob, shutil
source = "C:/Users/baba/Desktop/ PackDLMap /*RdEdg*.xml"  * は、任意の文字列
dest_dir = "C:/Users/baba/Desktop/RdEdg"
for file in glob.glob(source):    マッチしたファイルを１つずつ
   shutil.move(file, dest_dir)    移動

Tip

Save the options

To save the state of console’s widgets you have to close the Python Console from the close button. This allows you to save the geometry to be restored to the next start.
