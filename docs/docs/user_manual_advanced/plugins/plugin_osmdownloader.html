Aino

  • インストールと設定

Aino プラグインは、OpenStreemMap (OSM) のデータをダウンロードするためのツールですが、AI により曖昧な検索でデータを指定することができます。

インストールと設定

インストールが終わると、メニューバーのプラグインに、OSM Parser というメニューが追加されます。

サイトでアカウントを作成する必要があります。 Professional が $59、ビジネスが $349 のようです。

有料プランしかないようなので、ここまでしか試していません。
