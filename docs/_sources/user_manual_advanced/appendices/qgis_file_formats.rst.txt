QGS/QGZ - QGISプロジェクトファイル形式
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

**QGS** 形式は、QGISプロジェクトを保存するためのXML形式です。 
**QGZ** 形式は、QGSファイルとQGDファイルを含む圧縮（zip）アーカイブです。

* プロジェクトのタイトル
* プロジェクトのCRS
* レイヤツリー
* スナップ設定
* リレーション: マニュアルでは「関係」と訳されています。ただし、マニュアル中の他の箇所では「リレーション」と訳されています。
* マップキャンバスの範囲
* プロジェクトモデル
* 凡例
* mapviewドック（2Dおよび3D）
* 基になるデータセット（データソース）へのリンクを持つレイヤ、および範囲、SRS、結合、スタイル、レンダラ、ブレンドモード、不透明度などのその他のレイヤプロパティ。
* プロジェクトのプロパティ

QML - QGISスタイルファイル形式
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

QMLは、レイヤのスタイルを保存するためのXML形式です。

以下の URL に、日本の地理データ（国土数値情報や国勢調査）の QML ファイルがあります。
