.. _`processing.create_points`:

人口の数だけ点を作成
====================

国勢調査には人口 (JINKO) という属性があります。

この属性を使って、人口の数だけ点を作成するアルゴリズムを実行します。

まず、プラグイン jpdata を使って、国勢調査のデータをダウンロードしましょう。

.. _figure_qgis-processing-create-points-01:

.. figure:: img/qgis-processing-create-points-01.png
   :align: center

   jpData

次に、プロセシングで points などと検索し、「ポリゴン内にランダム点群 (inside)」を探します。

.. _figure_qgis-processing-create-points-02:

.. figure:: img/qgis-processing-create-points-02.png
   :align: center

   Processing

この画面で、少しわかりづらいですが、「点の数もしくは密度」の右側にあるアイコンをクリックします。

ここで、JINKO を選択します。

.. _figure_qgis-processing-create-points-03:

.. figure:: img/qgis-processing-create-points-03.png
   :align: center

   ポリゴン内にランダム点群 (inside)

実行を行うことで、点が作成されます。

.. _figure_qgis-processing-create-points-04:

.. figure:: img/qgis-processing-create-points-04.png
   :align: center

   ポリゴン内にランダム点群 (inside)

作成する点の多さに応じて時間がかかります。

.. _figure_qgis-processing-create-points-05:

.. figure:: img/qgis-processing-create-points-05.png
   :align: center

   JINKO の数だけ作成された点
