********
はじめに
********

jpData for QGIS は、世界中でよく使われている地理情報システム（GIS：Geographic Information System）である QGIS を、日本のデータ取得や日本での使用環境に適した機能を追加しています。

QGIS は、最もよく使われている地理情報システムの一つで、GIS とも呼ばれています。
GIS とは、位置情報や空間データと様々な情報をデジタル地図上に重ね合わせ、可視化・分析・管理するシステムです。
単なる地図の表示にとどまらず、データの管理や複数のデータを組み合わせた解析を行うことで、高度な意思決定をサポートします。

QGIS は、フリーソフトウェアです。誰でも自由に機能を追加することができます。
jpData も QGISに機能を追加するプラグインの一つです。
