Windows 版の注意点
==================

.. note::
	Windows は、デフォルトでは一部のファイルの拡張子が非表示にされています。GISデータでよく使うシェープファイルという形式では、拡張子だけが異なる同じ名前のファイルがたくさんあります。このため、拡張子は表示しておく方が良いです。
	ファイルエクスプローラの ... より、オプションを選択、「表示」タブから「登録されている拡張子は表示しない」を外しておきましょう。

Windows 版の QGIS は、注意点があります。

+ 「スタンドアロン」版と「OSGeo4W ネットワークインストーラー」版がある。
+ 複数の QGIS をインストールすることができる。古いバージョンは、なるべく削除しましょう。
とくに、QGIS バージョンごとに Python もインストールされるのですが、これが混乱の元になる時があります。

スタンドアロン版のパスは、C:\Program Files¥QGIS 3.34.7 などになります。
「OSGeo4Wネットワークインストーラー」版は、C:¥OSGeo4W フォルダにあります。

この中の bin フォルダ内にある grass83.bat で GRASS を、saga_gis.bat で SAGA を開くことができます。
