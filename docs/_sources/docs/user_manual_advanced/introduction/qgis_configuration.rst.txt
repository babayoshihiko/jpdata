
QGIS 設定
===================

.. only:: html

   .. contents::
      :local:
      :depth: 2

「設定」は、QGIS を使いやすくするためにさまざまな変更を加えることができます。

Windows では、メニューに「設定」があります。

macOSの場合、QGIS から Preferences を選択します。

フォントのエラー
^^^^^^^^^^^^^^^^^

「Open Sans フォントのインストールに失敗しました」と出る場合、以下の対処法があります。

- あらかじめ Open Sans をインストールしておく
-「設定」> フォント で、「フリーフォントを自動ダウンロード」をオフ
-「設定」> フォント で、フォント置換を設定する。

「フォント XXXX がこのOSでは利用できません」

-「設定」> フォント で、フォント置換を設定する。
