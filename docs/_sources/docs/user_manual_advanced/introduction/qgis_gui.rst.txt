QGIS GUI
========

.. only:: html

   .. contents::
      :local:

.. index::
   single: Main window

QGIS の Graphical User Interface (GUI) は、以下の画像の通りです。
とくに重要な①から⑤の用語を覚えておきましょう。

.. _figure_startup:

.. figure:: img/startup.png
   :align: center

   QGIS GUI with Alaska sample data

.. note::
   Your window decorations (title bar, etc.) may appear different
   depending on your operating system and window manager.

The main QGIS GUI (:numref:`figure_startup`) consists of five components /
component types:

#. :ref:`メニューバー <label_menubar>`
#. :ref:`ツールバー <sec_panels_and_toolbars>`
#. :ref:`パネル <sec_panels_and_toolbars>`
#. :ref:`マップビュー  <label_mapview>`
#. :ref:`ステータスバー <label_statusbar>`


これらに加えて、別ウィンドウがポップアップすることがあります。

ここで覚えておきたい用語があります。モーダルウィンドウです。
モーダルウィンドウとは、ウェブページ上のメインコンテンツの上に重ねて表示され、ユーザーの操作を一時的に遮断する小さなウィンドウやダイアログボックスのことです。
ユーザーの注意を特定の情報やアクションに集中させ、必要な情報を提示したり、確認を求めたり、フォーム入力を促したりする目的で使われます。
モーダルウィンドウは常に一番上にあり、これを閉じないと元のウィンドウを操作することができません。

QGIS では、「レイヤを追加」で出てくる「データソースマネージャー」や「属性ウィンドウ」、「印刷レイアウト」などは、モーダルウィンドウではありません。
画面が十分広い場合、メインウィンドウとこれらのウィンドウを並べて表示することも可能です。


メニューバー
------------

メニューバーは、QGIS の各種機能にアクセスするためのメニューが並んでいます。
しかし、QGIS は、昨日が膨大であり、メニューにすべての機能が表示されているわけではありません。

プラグインをインストールすると、メニューバーに新たなメニューが追加されることがあります。


ステータスバー
---------------

データが巨大で、描画に時間がかかる場合h、「レンダ」の左にあるチェックボックスをオフにしておきましょう。
こうすることで、毎回描き直しをしないで作業することができます。
