フォルダ設定
=========================

ここでは、まずは ``Shape Folder`` は設定してください。
ここで設定したフォルダにダウンロードしたファイルを保存します。

下記の動画では、プラグイン jpData のインストールから国勢調査表示までを 1:30 ほどで行っています。

.. raw:: html
   
   <iframe width="560" height="315" src="https://www.youtube.com/embed/DpV9DTfFroQ?si=D6OTPPHFqtOK6jTw" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>

* 実験的プラグインをチェック
* プラグインをインストール
* jpData を開く
* 保存先フォルダを選択
* 国勢調査をダウンロード・地図に追加


GIS データは巨大で、すぐに数 GB になります。
十分な余裕のあるディスクを選んでください。

macOS では、フォルダ名を ``data.noindex`` などのように ``.noindex`` とつけると、Spotlight 検索を行わなくなるため、CPU やディスクの節約になります。

Windows では、ファイルエクスプローラの...より、オプションを選択、「表示」タブから「登録されている拡張子は表示しない」を外しておきましょう。
GIS、特にシェープファイルでは拡張子を確認する必要があります。

シェープファイル (Shapefile) とは、ESRI 社の ArcGIS などの GIS で、1990 年代から 使われてきたファイル形式です。
「ファイル形式」と言いますが、実際には複数の拡張子を持つファイルから構成されています。

* .shp 地物のジオメトリデータが保存されています。
* .shx データに高速にアクセスするために使われています。
* .dbf 地物の属性値が保存されています。
* .prj (オプショナル) 測地系の情報が保存されいます。このファイルはなくても開くことができますが、QGIS は測地系がわからない状態で開きます。開いた後で QGIS 上で指定することはできます。

バックグラウンドでダウンロード
===============================

``バックグラウンドでダウンロードしない`` にチェックすると、ダウンロード中に QGIS 内では他の操作ができなくなります。
これは古いコーディングによるものなので、将来的には無くなる予定です。

ジオメトリをチェック
===============================

``ジオメトリをチェック`` は、デフォルトでチェックされています。
国土数値情報や国勢調査など ``地図を追加`` する際に、ジオメトリをチェックします。

ジオメトリは、ここで不正が見つかると交差などの処理の際にエラーが出ることがあります。
以下のコードで、 `layer` を渡すことで不正なジオメトリを数えます。

.. code-block:: python
   :linenos:

   from qgis.core import QgsVectorLayer
   if isinstance(layer, QgsVectorLayer):
      count = 0
      for feature in layer.getFeatures():
         if not feature.geometry().isGeosValid():
               count = count + 1

現在の実装は上になります。

他の可能性は、以下のようなものがあります。
あとの方ほど時間がかかる可能性があります。

* 数を数えず、最初の 1 件で止める
* Processing の ``native:checkvalidity`` を使う
* ``errors = geom.validateGeometry()`` を使う

さらに、修復まで行うことも可能です。

もう一つの方法は、あらかじめジオメトリをチェックしておき、ダウンロード情報などに記録しておくことです。
こうすることで、不正なジオメトリをチェックする必要はなくなります。
