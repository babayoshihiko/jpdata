This section describes the uniform styles that will be used throughout
this manual.

GUI の表記
---------------

The GUI convention styles are intended to mimic the appearance of the
GUI. In general, a style will reflect the non-hover appearance, so a
user can visually scan the GUI to find something that looks like the
instruction in the manual.

* メニュー(コンテキストメニューも含む): :menuselection:`Layer --> Add a Raster Layer` or
  :menuselection:`Settings --> Toolbars --> Digitizing`
* ツール: |addRasterLayer| :sup:`Add a Raster Layer`
* ボタン : :guilabel:`Save as Default`
* ダイアログのタイトル: :guilabel:`Layer Properties`
* タブ: :guilabel:`General`
* チェックボックス: |checkbox| :guilabel:`Render`
* ラジオボタン: |radioButtonOn| :guilabel:`Postgis SRID`
  |radioButtonOff| :guilabel:`EPSG ID`
* 数値を選択: |selectNumber|
* 文字を選択: |selectString|
* ファイルを探す: :guilabel:`...`
* 色を選ぶ: |selectColor|
* スライダー: |slider|
* テキストを入力: |inputText|

.. * Toolbox : \toolboxtwo{nviz}{nviz - Open 3D-View in NVIZ}

文字などの表記
----------------------------

本書は、プログラミングについても触れています。
以下は、主にプログラミングをする際の表記になります。

.. Use for all urls. Otherwise, it is not clickable in the document.

* ハイパーリンク: https://qgis.org
* 複数キーでの操作: Press :kbd:`Ctrl+B`, meaning press and hold the Ctrl
  key and then press the B key.
* ファイル名: :file:`lakes.shp`
* クラス名: **NewLayer**
* 関数名・メソッド名: *classFactory*
* サーバ: *myhost.de*
* 入力するテキスト: ``qgis --help``

.. * Single Keystroke: press \keystroke{p}
.. * Name of a Field: \fieldname{NAMES}
.. * SQL Table: \sqltable{example needed here}

Lines of code are indicated by a fixed-width font:

::

    PROJCS["NAD_1927_Albers",
      GEOGCS["GCS_North_American_1927",

OS 特有の事項
------------------------------

本書は、主に Windows ユーザを対象にしていますが、
著者自身が macOS ユーザです。
このため、スクリーンショットはだいたい macOS でとっています。

それぞれの OS に固有のことについては、以下のように注記を記しています。

* |nix| Linux など Unix 系。ほとんどありません。
* |win| Windows 固有の注意事項。
* |osx| macOS 固有の注意事項。
