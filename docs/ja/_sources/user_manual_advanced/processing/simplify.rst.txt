簡素化
======

簡素化 (Simplify) は、QGIS の基本ツールにあります。
ラインストリングやポリゴンの頂点の数を減らし、簡素化します。
ファイルサイズが小さくなったり、交差 (空間結合) が高速化するなどの利点があります。

ただし、QGIS 基本ツールの簡素化は、問題があります。トポロジが維持されないため、隣接するポリゴンの境界が一致しなくなります。

トポロジーを維持したまま簡素化するのは GRASS の v.generalize です。

プラグインで、Processing に加えて、GRASS GIS Provider をインストール・有効化する必要があります。
GRASS 本体は、通常はインストールされているはずです。

なお、異なるレイヤを簡素化する場合、あらかじめマージしておく必要があります。
